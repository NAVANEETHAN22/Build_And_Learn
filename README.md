# Immersive Learning and Construction Beyond Reality through Advanced Virtual Reality

An educational Virtual Reality (VR) application developed using Unreal Engine 5, Blender, and Convai AI. The project provides an immersive environment where users can learn computer hardware concepts by assembling virtual components such as keyboards, laptops, CPUs, and other devices through interactive VR experiences.

## Project Overview

This project aims to bridge the gap between theoretical learning and practical experience through Virtual Reality. Users can interact with realistic 3D models, assemble computer hardware components, and receive real-time guidance from an AI-powered assistant.


## Features

- Immersive VR learning environment
- Interactive assembly of computer hardware components
- Realistic 3D models created using Blender
- AI-powered guidance using Convai
- Hand-tracking and object interaction
- Educational modules for beginners and advanced learners
- Real-time feedback during object construction
- Built using Unreal Engine 5 and OpenXR

## Technologies Used

| Technology | Purpose |
|------------|----------|
| Unreal Engine 5 | VR application development |
| Blender | 3D modeling |
| Convai | AI-powered interaction |
| OpenXR | VR hardware compatibility |

The application consists of three major components:

- Unreal Engine for VR interaction and gameplay.

- Convai AI for real-time conversational assistance and educational guidance.
